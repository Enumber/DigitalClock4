Bundled textures — provenance
=============================

These 210 texture images are **not** our work and are **not** part of the
Digital Clock 4 source tree. They are the texture set bundled with the official
Digital Clock 4 binary distributions, taken verbatim from the official Windows
package `digital_clock_4-win_portable.zip` (4.7.9, 2020-12-20).

They carry no attribution or licence information of their own, and upstream
never stated terms for them separately from the program's GPL-3. We therefore
make no licence claim about them; we redistribute them unmodified so the Linux
build offers the same texture picker as the Windows build.

The program works fine without this folder — the texture picker simply comes up
empty. See `../skins/README.md` for the same caveat applied to the skins, and
for how to request removal.
